# Health Insurance Premium Prediction

## Overview
This project predicts health insurance premiums using machine learning algorithms based on customer information such as age, gender, BMI, and health insurance coverage.

## Problem Statement
Build a regression model to predict the premium amount for health insurance customers.

## Dataset Features
- Age
- Gender
- Height
- Weight
- Health Insurance Cover

## Target Variable
- Premium

## Machine Learning Algorithms Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- K-Nearest Neighbors (KNN)

## Hyperparameter Tuning
- GridSearchCV
- RandomizedSearchCV

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data Collection
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Feature Selection
5. Model Building
6. Hyperparameter Tuning
7. Model Evaluation
8. Prediction
